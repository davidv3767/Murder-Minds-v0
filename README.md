# Murder-Minds-v0
This is a Campfire HackClub project made by David Vuddandam & Jayden Wainwright.
This game is about a murderer, you, trying to evade getting caught in the mineshaft.
The two ways to play are simulation (kind of like a console game, where you type in numbers that resemble choices) & side scroller (graphics version of the game, where you have to escape).

Please note this game is not fully completed as there are still many bugs to get to a v1 of this game. However, we have a big idea in mind (while brainstorming this as a console-only game) that will hopefully be impleted in the future by David.

Controls:
Simulation -> Number Keys
Side Scroller -> space bar (to mine), e (elevator/door), b (currently in a hidden room), & f (to murder the NPCs)
